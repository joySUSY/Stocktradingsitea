## ⚠️ 常见问题

### Q1: 提示 "pip 不是内部或外部命令"
**解决：** Python未正确安装或未添加到PATH
- Windows: 重新安装Python，勾选"Add to PATH"
- 或使用完整路径：`C:\Python\python.exe -m pip install -r requirements.txt`

### Q2: 安装依赖时报错
**解决：** 升级pip
```bash
python -m pip install --upgrade pip
```
然后重新安装依赖

### Q3: 搜索显示"本地数据库"
**原因：** 后端服务未启动或未连接
**解决：** 
1. 检查后端是否运行
2. 检查 http://localhost:3001 是否可访问
3. 查看后端控制台是否有错误

### Q4: 显示"未找到匹配的股票"
**原因：** 可能是非交易时间，部分股票无数据
**解决：** 尝试搜索主流股票，如"茅台"、"平安"等

### Q5: Token失效
**解决：** 
1. 访问 https://tushare.pro
2. 登录查看token状态
3. 如需更新，修改 `backend/stock_api_server.py` 第18行
