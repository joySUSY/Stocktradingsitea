## 🎯 超级简单的启动方式

### Windows用户（推荐）：

1. **双击运行** `backend/start_server.bat` 文件
2. 等待服务启动（首次会自动安装依赖）
3. 看到"Running on http://127.0.0.1:3001"即可使用
4. **保持窗口运行，不要关闭**

### Mac/Linux用户：

1. 打开终端，进入backend文件夹
   ```bash
   cd backend
   chmod +x start_server.sh
   ./start_server.sh
   ```
2. 等待服务启动
3. **保持终端运行，不要关闭**

## 🔧 如果遇到问题

### 问题1: 双击.bat文件没反应
**解决：** 右键 → 以管理员身份运行

### 问题2: 提示找不到Python
**解决：** 
1. 下载安装Python: https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"

### 问题3: 搜索时显示"本地数据库"
**解决：** 
1. 确保后端服务正在运行
2. 浏览器访问 http://localhost:3001 测试
3. 如果无法访问，重启后端服务

### 问题4: 安装依赖失败
**解决：**
```bash
python -m pip install --upgrade pip
pip install flask flask-cors tushare pandas
```

## 💡 使用技巧

1. **每天使用前**：启动后端服务
2. **使用完毕后**：按 Ctrl+C 停止服务
3. **最佳使用时间**：交易日 9:30-15:00
4. **数据更新**：自动缓存60秒，减少API调用
5. **自选股备份**：定期导出浏览器书签备份
